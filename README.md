# automated-ci-cd-pipeline
Developed an automated CI/CD pipeline using Jenkins, Docker, Kubernetes, and AWS to automate build, test, and deployment processes. The system reduces manual effort and enables faster, reliable, and scalable software delivery using cloud-based deployment.
Here it is
Automated Cloud-Based CI/CD Pipeline
📌 Overview

This project is an automated CI/CD pipeline built using DevOps tools to streamline software development and deployment. It automates the process of building, testing, and deploying applications using cloud infrastructure.

⚙️ Features
Automated build, test, and deployment process
Continuous Integration and Continuous Deployment (CI/CD)
Docker-based containerization
Kubernetes-based deployment
Cloud deployment using AWS
Reduces manual work and deployment errors
🛠️ Technologies Used
Jenkins
Docker
Kubernetes (EKS)
AWS
Git & GitHub
🔄 Workflow
Code is pushed to GitHub
Jenkins triggers the pipeline
Application is built and tested
Docker image is created
Application is deployed to Kubernetes cluster
Runs on AWS cloud
